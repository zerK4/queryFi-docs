### `limit` Method Documentation

The `limit` method specifies the sorting order for query results. It supports **single columns**, **multiple columns**, and **key-value pairs** with optional sorting directions (`asc` or `desc`).  
<Note type="note" title="Note">
  **After finishing building your query attach `.build()` to generate it.**
</Note>
---

## Method Signature

```typescript
limit(value: number): this;
```

- **`value`**: The number of records to limit the query to. This must be a positive integer.

---

### Return Type

- **`this`**: The method returns the current instance, allowing for method chaining.

The `limit` method allows you to control the number of records returned by your query. It is useful for pagination, performance optimization, or simply limiting the dataset for a specific use case. By passing a positive integer, you can easily control the query results.
